# AI Short Drama Production Workflows

English | [简体中文](README.zh-CN.md)

A practical workflow repository for AI short drama production, from idea and worldbuilding to character systems, scripts, storyboards, visual assets, and short video output.

This repository focuses on the production chain behind AI short drama creation. It is designed for creators, studios, and content teams who want to understand the full workflow rather than only the final video generation step.

## Why AI Short Drama Needs A Production Workflow

AI short drama creation is not just text-to-video generation.

A complete production workflow usually includes:

- idea refinement
- concept validation
- worldbuilding
- character system design
- episode outline
- scriptwriting
- character visualization
- scene visualization
- storyboard planning
- camera movement
- final short video output

## Core Production Stages

```text
Idea
-> concept
-> worldbuilding
-> character system
-> outline
-> script
-> visual assets
-> storyboard
-> camera movement
-> short drama video
```

## Recommended Production Principles

1. Validate the concept before generating assets.
2. Build character consistency before storyboarding.
3. Separate worldbuilding, scriptwriting, and visual asset generation.
4. Treat storyboard as a production plan, not just a visual preview.
5. Keep human review points between major AI-generated stages.
