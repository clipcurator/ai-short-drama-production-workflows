# Worldbuilding Workflow

Worldbuilding gives a short drama its rules, tone, and story pressure.

## Define

- time period
- location
- social rules
- visual tone
- core conflict
- story constraints

## Keep it production-friendly

Good AI short drama worldbuilding should not be too vague. It needs to guide:

- costumes
- scene design
- character behavior
- dialogue style
- visual continuity

## Output checklist

- setting is clear
- rules are usable
- conflict is visible
- tone is consistent
- scenes can be generated from the world description
