# Script to Assets

Turning a script into assets means extracting everything the production system needs before video generation.

## Extract from script

- characters
- locations
- props
- time of day
- emotional beats
- visual moments
- dialogue-heavy scenes

## Asset types

- character visuals
- scene concepts
- props
- style references
- shot prompts

## Review

Check whether the assets support the script rather than simply looking good in isolation.
