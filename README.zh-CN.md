# AI Short Drama Production Workflows

[English](README.md) | 简体中文

这是一个面向 **AI 短剧生产、AI 影视创作、灵感到剧本、剧本到资产、分镜到视频** 的工作流仓库。

这个仓库重点整理完整生产链路，而不是只讲最后一步视频生成。

## 为什么 AI 短剧需要生产工作流

AI 短剧创作不只是 text-to-video。

完整流程通常包括：

- 灵感提纯
- 概念验证
- 世界观构建
- 人物系统设计
- 分集大纲
- 剧本创作
- 角色可视化
- 场景可视化
- 分镜规划
- 镜头运动
- 短剧视频输出

## 核心生产阶段

```text
灵感
-> 概念
-> 世界观
-> 人物系统
-> 大纲
-> 剧本
-> 视觉资产
-> 分镜
-> 镜头运动
-> 短剧视频
```

## 推荐生产原则

1. 先验证概念，再生成资产
2. 先建立人物一致性，再做分镜
3. 世界观、剧本和视觉资产要分阶段处理
4. 分镜不是预览图，而是生产计划
5. 每个关键 AI 生成阶段之间都保留人工审核
